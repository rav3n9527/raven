PipBoy v1.0.0 · iOS 15+

多任务增强插件 — 在应用切换器中注入毛玻璃控制面板，一键关闭所有后台应用，实时显示内存和存储状态，支持下滑锁定白名单。

功能特性

🗑 关闭所有后台
📊 内存 / 存储监控
🔒 下滑锁定白名单
🎵 自动保护播放中应用

使用方法

1. 打开多任务切换器
2. 底部毛玻璃面板显示系统信息
3. 点击 关闭所有 一键清后台
4. 在卡片上 向下滑动 锁定 / 解锁
